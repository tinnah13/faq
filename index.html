What is jQuery and why is it used?

⌄

jQuery is a fast, lightweight JavaScript library that simplifies HTML DOM traversal, event handling, and animation. It wraps complex browser APIs into short, readable methods — what used to take 10 lines of vanilla JS can often be done in one line with jQuery.

What jQuery method powers this toggle?

⌄

slideToggle() animates the height of an element between hidden and visible. It automatically switches between slideDown() (expand) and slideUp() (collapse) on each call, creating a smooth accordion effect.

How does jQuery handle events?

⌄

jQuery uses .on(event, handler) or shorthand methods like .click() to bind event listeners. Inside the handler, $(this) refers to the element that triggered the event, making it easy to find related siblings or parents.

What is the difference between hide() and slideUp()?

⌄

hide() instantly sets display: none with no animation. slideUp() gradually collapses the element's height to zero before hiding it, producing a smooth animated effect that feels more polished in UI components like accordions.

What is the $(document).ready() pattern?

⌄

It ensures your jQuery code runs only after the full HTML document has loaded and the DOM is ready. The modern shorthand is simply $(function() { ... }). Without it, scripts may try to select elements that don't exist yet.

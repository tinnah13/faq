Features

Click a question to expand/collapse its answer
Only one answer open at a time (accordion behavior)
Expand all / Collapse all button
Smooth slide animation via jQuery


jQuery Methods Used

slideDown() — animates an element open
slideUp() — animates an element closed
slideToggle() — toggles between the two
.on('click', fn) — binds click events
$(this).closest() / .find() — DOM traversal
$(function() { }) — runs code after DOM is ready
